The callback invoked when a client set a goal to the {\ttfamily planner} server. This function will return a list of random points (i.\+e., the plan) when the fist point is the current robot position (retrieved from the {\ttfamily my\+\_\+helper} node), while the last point is the {\ttfamily goal} position (given as input parameter). The plan will contain a random number of other points, which spans in the range \mbox{[}{\ttfamily self.\+\_\+random\+\_\+plan\+\_\+points\mbox{[}0\mbox{]}}, {\ttfamily self.\+\_\+random\+\_\+plan\+\_\+points\mbox{[}1\mbox{]}}). To simulate computation, each point is added to the plan with a random delay spanning in the range \mbox{[}{\ttfamily self.\+\_\+random\+\_\+plan\+\_\+time\mbox{[}0\mbox{]}}, {\ttfamily self.\+\_\+random\+\_\+plan\+\_\+time\mbox{[}1\mbox{]}}).
