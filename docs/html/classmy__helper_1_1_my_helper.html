◆ is_battery_low()

def my_helper.MyHelper.is_battery_low (   self)

This function get the state of _battery_low variable.

Parameters
None
Returns
_battery_low

Get the state variable encoded in this class that concerns the battery level. The returning value will be True if the battery is low, False otherwise. Note that the node using this class might exploit the reset_battery function to improve robustness. Also note that this function should be used when the mutex has been acquired. This assures the synchronization with the threads involving the subscribers and action clients.
